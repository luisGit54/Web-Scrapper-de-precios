# 🕷️ Web Scraper de Precios Básico

Script en Python que extrae nombres y precios de productos de una página web de prueba y los guarda en un archivo CSV.

## 🎯 Objetivo del Proyecto
Demostrar habilidades básicas de:
- Web scraping con `BeautifulSoup`
- Peticiones HTTP con `requests`
- Manejo de archivos CSV
- Control de errores con `try/except`
- Uso de bucles y estructuras de datos

## 🚀 Instalación y Uso

1. Clona este repositorio:
```bash
git clone https://github.com/tu-usuario/price-scraper-basic.git
cd price-scraper-basic

2. Instala las dependencias :bash
pip install -r requirements.txt

3. Ejecuta el script
python price_scraper.py

4. Revisa el archivo productos.csv generado
